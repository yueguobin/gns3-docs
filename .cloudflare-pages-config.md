# Cloudflare Pages 配置

## 构建设置 (Build settings)

- **Framework preset**: `None`
- **Build command**: `yarn build`
- **Build output directory**: `build`
- **Root directory**: `/` (默认)

## 环境变量 (Environment Variables)

不需要额外环境变量

## 分支设置

- **Production branch**: `deploy/yueguobin-account` (或 `master`)
- **Preview branches**: 启用（为所有 PR 创建预览）

## 自动部署

- 推送到 `deploy/yueguobin-account` 分支 → 自动部署到生产环境
- 推送到其他分支 → 创建预览部署
